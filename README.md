## Technology stack
- Kotlin
- Multi module
- Single Activity
- MVVM
- Jetpack ViewModel + LiveData
- Jetpack Navigation
- Dagger
- Room
- Coroutines
- Widgets
- Custom Pie Chart View
- Custom Bar Chart View
- Recycler with custom Adapter Delegates
- DiffUtils with payloads
- Drag and Drop
- Shared ViewModels
- Snackbars with Action
- Gradle Kotlin DSL
