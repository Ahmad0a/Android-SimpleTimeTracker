## Technology stack
- Kotlin
- Androidx
- MVVM
- Jetpack ViewModel + LiveData
- JetPack Navigation
- Dagger
- Room
- Coroutines
- Gradle Kotlin DSL
- Custom Pie Chart View
- Recycler with custom Adapter Delegates,
- DiffUtils with payloads
