## Technology stack
- Kotlin
- Multi module
- Single Activity
- MVVM
- Jetpack ViewModel + LiveData
- Jetpack Navigation
- Dagger
- Room
- Room migrations
- Coroutines
- Several Widgets
- Notifications
- Custom Pie Chart View
- Custom Bar Chart View
- Recycler with custom Adapter Delegates
- DiffUtils with Payloads
- Drag and Drop
- Shared ViewModels
- Snackbars with Action
- Gradle Kotlin DSL
- Database backup and restore
- Database export to csv
- Dark mode
- UI tests
